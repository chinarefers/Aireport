## Aireport 轻空气

一个使用了最精简的代码,最新的iOS11 API,启动速度最快的空气质量App.
*  没有使用任何第三方框架,追求做到最精简,安装包最小,启动速度最快
*  使用了Swift4.0 Encoding and Decoding, model转换效率大增
*  Background fetch技术, 每隔一小时后台刷新数据, 将空气质量数据显示在Icon上,无需频繁打开App
*  添加widget
*  数据来源: waqi.info
